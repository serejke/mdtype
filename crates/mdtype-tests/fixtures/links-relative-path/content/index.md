## Setup

See [the setup guide](setup.md) for prerequisites.

A [missing target](does-not-exist.md) should diagnose.

A link to a real file's [valid heading](setup.md#install) is fine; a
[bad anchor](setup.md#nonexistent) is not.

External [HTTPS](https://example.com) links are skipped, as is [mail](mailto:hi@example.com).

A same-file [good anchor](#setup) is fine; a same-file
[bad anchor](#nope) is not.
