# mdtype-reporter-json

Structured JSON reporter for [`mdtype`](https://github.com/serejke/mdtype). Emits a versioned payload — field shape and key names are the public contract. Pretty-prints when stdout is a tty, compact otherwise.

See [`docs/json-schema.md`](https://github.com/serejke/mdtype/blob/main/docs/json-schema.md) for the contract reference and the versioning rule.
